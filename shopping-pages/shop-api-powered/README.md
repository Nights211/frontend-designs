# API-Powered Shop

E-commerce page that fetches products from the backend API.

## Features

- Fetches products from FastAPI backend
- Displays product details (name, price, description, stock)
- Responsive grid layout
- Category badges

## Requirements

The backend must be running for this example to work.

Start the backend:
```bash
./launch.py backend
```

Then launch this example:
```bash
./launch.py shopping shop-api-powered
```

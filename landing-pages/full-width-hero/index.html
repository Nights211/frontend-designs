Installation

Install the Neural SDK using pip:

pip install neural-sdk

Or using npm for JavaScript:

npm install @neural/sdk

Authentication

Get your API key from the dashboard and authenticate:

import neural
client = neural.Client(api_key="your_api_key_here")

Examples

Generate text with GPT-4:

response = client.generate(
    prompt="Write a haiku about AI",
    model="gpt-4",
    max_tokens=50
)
print(response.text)

API Endpoints

POST /v1/generate - Generate text

POST /v1/images - Generate images

GET /v1/models - List available models

Available Models

API Reference

Generate Text

POST /v1/generate

Request Body

{
  "prompt": "string",
  "model": "gpt-4",
  "max_tokens": 100,
  "temperature": 0.7
}

Response

{
  "id": "gen_123abc",
  "text": "Generated text...",
  "tokens_used": 45,
  "model": "gpt-4"
}

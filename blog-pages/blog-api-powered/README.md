# API-Powered Blog

A blog that fetches content from the backend API.

## Features

- Fetches posts from FastAPI backend
- Displays post metadata (category, date, author)
- Clean card-based layout

## Requirements

The backend must be running for this example to work.

Start the backend:
```bash
./launch.py backend
```

Then launch this example:
```bash
./launch.py blog blog-api-powered
```
